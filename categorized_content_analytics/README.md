#### Цель проекта:
Исследовать категории контента для выявления закономерностей и оптимизации ленты рекомендаций.

#### Описание проекта
Данные анализа были синтетически сгенерированы для сохранения конфиденциальности, но исследование основано на реальной задаче и повлияло на работу нескольких отделов.

Проект выполнялся в рамках стажировки: данные были выгружены для отдела маркетинга, исследование проводилось без формального требования использования результатов. Несмотря на это, выводы исследования были переданы в отдел маркетинга, затем распространены смежным подразделениям и дошли до директора по развитию бизнеса.
#### Результаты
- Были изменены правила разметки контента.
- Были скорректированы подходы к настройке ленты рекомендаций.
- Отдел аналитики начал расследование причин аномалий в выгруженных данных.
